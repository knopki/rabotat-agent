Аргументы коммандной строки:
  --secret <string>    Общий секрет для общения с сервером (обязательно)
  --submit <string>    URL серверной части для отправки данных (обязательно)
  --step   <int>       Время в секундах между записями активных окон (по-умолчанию: 1)
  --idle   <int>       Время в секундах неактивности, после которого перестают делаться записи (по-умолчанию: 60)
